# My Blog

Welcome to my personal blog! This project is a simple static blog that showcases various entries on topics of interest. Below is an overview of the project structure and how to set it up.

## Project Structure

```
my-blog
├── index.html        # Main homepage of the blog
├── entrada1.html     # First blog entry
├── entrada2.html     # Second blog entry
├── entrada3.html     # Third blog entry
├── styles
│   └── main.css      # Styles for the blog
├── images
│   ├── blog1.jpg     # Image for the first blog entry
│   ├── blog2.jpg     # Image for the second blog entry
│   └── blog3.jpg     # Image for the third blog entry
└── README.md         # Documentation for the project
```

## Setup

1. Clone the repository or download the project files.
2. Open the `index.html` file in your web browser to view the homepage.
3. Navigate to the individual blog entries by clicking the links provided on the homepage.

## Customization

Feel free to modify the HTML files to add your own content, change the styles in `main.css`, or replace the images in the `images` directory with your own.
